What is SamanirA ?

SamanirA is an easy HTML / CSS / JS framework for "Front-end Developers" who wants to quickly build a project from 0% to 100% without wasting time on defining css classes or some javascript functions and plugins.
Created by Saman Rashidi & Samira Salimian

Why SamanirA ?

We have done more than 50 front-end projects such as Website, Landing Page, CMS, CRM and … in 4 years then we noticed that we need a light and comprehensive css framework to work with and it will have so many benefits for our projects:

  • It is open source and easy to understand for changing values and classes based on your project's template
  • It has the same arrangement for all css properties
  • Most of the css classes and javascript functions that you need in projects are defined
  • Back-end developers will love you for your code discipline
  • Your folders, codes and classes will be same for all projects

Where to begin ?

  1. Download from Github
  2. Unzip in your project's folder
  3. Start coding your project

- Put each label and form item (input/textarea/select) in a <div> with class of field-holder

- If you want to add icons to your input just wrap it in a <div> with class of form-field-holder and then put your icon in .field-icon

- .reverse-input will change the direction of an input

- .textarea-word-tracker will make a textarea with the counting number function in samanira.js

- .samanira_secure will be for an input with the type of text and is for preventing form from spamming with bots in front-end side with a function in samanira.js
